KAIROS — Static single-viewport site

To run locally:

```bash
npm install
npm run dev
```

This site is intentionally minimal: a single, non-interactive viewport with a WebGL background and centered content.

Deployments: GitHub Actions workflow deploys to GitHub Pages. Add repository `main` branch and set Pages to use `gh-pages` or the provided workflow output. Ensure `CNAME` is configured in your DNS.
